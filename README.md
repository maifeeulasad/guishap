# guishap - জনসাধারন সুলভ বাংলা প্রোগ্রামিং ল্যাঙ্গুয়েজ

 - Variable Declaration :
   - Just Declaration :
	```
	সংখ্যা x;
	স্ট্রিং x;
	```
   - Declaration & Initialization :
	```
	সংখ্যা x = 123;
	স্ট্রিং x = "123";
	ধ্রুবক সংখ্যা x = 123;
	ধ্রুবক স্ট্রিং x =  "123";
	সংখ্যা x = y;
	স্ট্রিং x = y;
	ধ্রুবক সংখ্যা x = y;
	ধ্রুবক স্ট্রিং x = y;
	```
